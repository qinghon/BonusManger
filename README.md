# BonusManger
博纳云管理工具守护程序

## 安装
```bash
# 下载
curl -fsSL -o /tmp/install.sh https://raw.githubusercontent.com/qinghon/BonusManger/master/install.sh
# 安装
bash /tmp/install.sh
```
### 卸载
```bash
bash /tmp/install.sh remove
```
